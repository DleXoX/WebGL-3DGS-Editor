# WebGL-3DGS-Editor
WebGL 3D Gaussian Splat Editor

在 [antimatter15的3dgs viewer](https://antimatter15.com/splat/) 基础上稍加修改而成，提供简单的编辑功能，目前只支持编辑ply格式的文件。代码写得一坨，有空再改吧。
